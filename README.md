# Proyecto: Publicidad en Muros - All Vista Reforma

Este repositorio concentra el código y los recursos del sitio web utilizado para documentar y exponer las evidencias relacionadas con la **publicidad en muros** dentro del condominio *All Vista Reforma*.

## Objetivo

Proporcionar a los condóminos un acceso centralizado, claro y ordenado a:

- Evidencias fotográficas.
- Documentos legales relevantes.
- Licencia de publicidad registrada ante SEDUVI.
- Presentación e infografía compartidas en Asamblea.

## Estructura del proyecto

```bash
.
├── assets/
│   ├── docs/                # Documentos PDF e imágenes de referencia
│   ├── img/                 # Imágenes como favicon y recursos gráficos
├── index.html               # Página principal del sitio
├── style.css                # Estilos globales del sitio
├── script.js                # Lógica para interacciones (copiar al portapapeles)
└── README.md                # Este archivo
```

## Tecnologías utilizadas

- **HTML5** para la estructura del sitio.
- **CSS3** con variables y media queries para un diseño adaptable.
- **JavaScript** vanilla para la funcionalidad de copiado.
- **Font Awesome** para el uso de iconos.
- **GitHub Pages** para la publicación del sitio.

## Contenido principal

El sitio se divide en secciones:

1. **Inicio**: introducción y enlaces a presentación e infografía.
2. **Evidencias**: fotografías de muros y medidores que prueban el uso de áreas comunes.
3. **Licencia SEDUVI**: número de licencia con acceso directo a la plataforma oficial.
4. **Marco legal**: extractos relevantes de la escritura, reglamento, ley de condominio y Código Penal CDMX.

## Publicación

Este sitio se despliega mediante **GitHub Pages**. Para publicarlo:

1. Habilitar GitHub Pages desde la configuración del repositorio.
2. Seleccionar la rama principal (`main`) como fuente.
3. Acceder al sitio desde la URL generada por GitHub Pages.

## Notas

- Los documentos íntegros se encuentran en la administración del condominio; aquí solo se muestran extractos.
